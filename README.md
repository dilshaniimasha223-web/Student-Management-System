# Student-Management-System

## Data Structures and Algorithms (ICT 143-2)
### Practical Assignment

## Project Overview

Student Management System is a console-based Java application developed to manage student records efficiently. The system demonstrates the practical use of data structures and algorithms to solve a real-world problem.

The application allows users to add, view, search, update, and delete student records. It also manages student registration requests using a Queue data structure.

---

## Features

- Add new students
- View all students
- Search students by ID
- Update student information
- Delete student records
- Store deleted students history
- Add registration requests
- Complete student registrations

---

## Data Structures Used

### 1. ArrayList
**Purpose:**  
Used to store and manage student records dynamically.

**Operations:**
- Add student
- View students
- Search students
- Update students
- Delete students


### 2. Stack
**Purpose:**  
Used to store deleted student records.

**Operations:**
- Push deleted students into history


### 3. Queue
**Purpose:**  
Used to manage registration requests.

**Principle:**
FIFO (First In First Out)

**Operations:**
- Add registration request (Enqueue)
- Complete registration (Dequeue)

---

## Algorithms Used

### Linear Search Algorithm
Used to search students by Student ID.

### Traversal Algorithm
Used to display all student records.

### Insert Algorithm
Used when adding new students.

### Delete Algorithm
Used when removing student records.

### FIFO Queue Algorithm
Used for processing registration requests.
